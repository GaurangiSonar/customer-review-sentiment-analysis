# Customer Review Sentiment Analysis

## Overview
A machine learning project that analyzes customer reviews and predicts sentiment as Positive, Negative, or Neutral.

## Features
- Text preprocessing and cleaning
- Sentiment classification
- Machine learning-based prediction
- Interactive web interface using Flask

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF
- Flask
- HTML/CSS

## Dataset
Flipkart Product Review Dataset
Approximate Size: 189,874 reviews

## Models Used
- Logistic Regression
- Naive Bayes

## Project Structure
- app.py
- model files (.pkl)
- dataset
- templates
- static files
